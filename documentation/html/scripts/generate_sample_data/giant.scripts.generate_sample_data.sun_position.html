giant.scripts.generate_sample_data.sun_position¶

giant.scripts.generate_sample_data.sun_position(date)[source]¶

Return the a priori inertial location of the sun.

Note

This is just a simulation position, it has not real physical correspondence

Parameters

date (datetime.datetime) – the date the position is requested

Returns

the position vector from the SSB to the sun in the inertial frame

Return type

numpy.ndarray
