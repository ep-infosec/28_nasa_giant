download_ucac¶

giant.catalogues.ucac:

giant.catalogues.ucac.download_ucac(target_directory)[source]¶

This function downloads the UCAC4 catalogue from vizier to the target directory.

This is done over ftp. It requires an active internet connection that can connect to cdsarc.u-strasbg.fr

Warning

This download will take a long time and use up approximately 9 GB of space.

Warning

This download can only verify the integrity of the files through md5 sums which are insecure. While the vizier service is trusted, use this function at your own risk

Parameters

target_directory (pathlib.Path) – the directory to save the UCAC catalogue to
