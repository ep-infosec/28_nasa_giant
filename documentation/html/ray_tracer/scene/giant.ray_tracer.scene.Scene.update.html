Scene.update¶

giant.ray_tracer.scene:

Scene.update(image, corrections=None)[source]¶

This method changes the scene to reflect the time specified by image optionally using corrections of “LT” (light time), “S” (stellar aberration”, or “LTPS” (light time and aberration).

Parameters

  • image (giant.image.OpNavImage) – The image from the camera you want to update the scene for as an OpNavImage

  • corrections (Optional[giant.ray_tracer.scene.CorrectionsType]) – A flag specifying which corrections to use or None to use the corrections specified for each object in the scene.
