Goddard Image Analysis and Navigation Tool¶

GIANT is an API for performing various kinds of Optical Navigation in simple python scripts. It provides all of the necessary tools to be able to easily perform most OpNav tasks including geometric camera calibration, attitude estimation, unresolved body center finding, resolved body center finding, limb based navigation, surface feature navigation, image planning, and much more.

If you’re just getting started with GIANT, check out the installation and getting started pages for some quick tutorials on how to get up and running. If you’re familiar with GIANT then you can go straight to the API reference.
