giant.ufo.visualizer¶

This module implements functions for viewing the results of UFO identification.

Use¶

There is currently a single user function in this module, show_detections(). Simply supply the detections DataFrame and the Camera object to this function and it will show the results for you to examine (alternatively it can save the plots to files). We hope to add more functionality to this module in the future.

Function

show_detections

This function plots possible UFO detections for each turned on image in camera over top of the image.
